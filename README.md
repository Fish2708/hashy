# hashy
Generating and Checking Hashes
***
## Currently Supported Algorithms
  - md5 
  - sha256
  - sha512
  - b2
  - b3
  - xxh32
  - xxh64
  - xxh128
## Usage
#### Generating:
  - hashy gen "text" -algo
#### Checking:
  - hashy check "text" "hash" -algo
#### Flags
  - -o Output Generated Hash to ./output.txt
